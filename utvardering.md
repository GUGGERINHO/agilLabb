# Utvärdering

## 1. Vad vi byggde

Vi byggde en enkel webshop utifrån våra user stories — från inloggning och produktöversikt till varukorg och utcheckning. Varje story implementerades stegvis och på egen branch.

## 2. Hur samarbetet fungerade

Vi arbetade med mobprogrammering i hela projektet. Det fungerade väldigt smidigt och gjorde att vi hela tiden var överens om riktning och lösningar.

## 3. Hur vi arbetade med Git

Vi följde git-guiden och arbetade med feature-branches, pull requests och kodgranskning före merge. Varje user story fick sin egen branch som slås ihop i rätt ordning.

## 4. Vad som fungerade bra

Kommunikationen var stark genom hela projektet. Vi delade idéer kontinuerligt, snabbt anpassade oss efter varandras arbetsätt och löste problem effektivt tillsammans.

## 5. Vad vi hade gjort annorlunda

Vi hade en merge-konflikt lokalt när vi separerade brancherna 003-Checkout och 004-Cart. Med bättre koordinering hade vi undvikit detta — antingen genom att hålla dessa på samma branch eller synka tidigare.
