# Planning

## Produktidé

Enkel webshop med inloggning, produktöversikt, varukorg och utcheckning.

## Backlog

1. **Inloggning** — Som kund vill jag kunna logga in.
2. **Produktöversikt** — Som kund vill jag se alla tillgängliga produkter med beskrivningar och alternativ.
3. **Produktdetaljer** — Som kund vill jag kunna se detaljerad information om produkter.
4. **Varukorg** — Som kund vill jag ha en varukorgssida som visar mina valda varor.
5. **Utkcheckning** — Som kund vill jag ha en utcheckningssida där jag kan ange kortuppgifter och adress.
